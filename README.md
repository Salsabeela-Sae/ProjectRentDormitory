## OOP Project - Rent Dormitory System(First Year)
ระบบนี้เป็น **โปรแกรม Console Application** สำหรับจัดการการจองหอพัก โดยผู้ใช้งานสามารถ:

- ดูรายการห้องว่าง
- จองห้องพัก
- จัดการข้อมูลลูกค้า
- คำนวณค่าเช่ารวมสิ่งอำนวยความสะดวก

ทั้งหมดสามารถทำได้ผ่าน **หน้าต่างคอนโซล** โดยไม่มี UI แบบกราฟิก

---

## คุณสมบัติหลัก

1. แสดงรายการห้องพักทั้งหมดพร้อมราคาและสถานะ (ว่าง/จองแล้ว)
2. จองห้องพักโดยกรอกข้อมูลลูกค้า
3. เพิ่มสิ่งอำนวยความสะดวกเสริม เช่น Fitness, สระว่ายน้ำ, Co-working Space
4. คำนวณค่าเช่าห้องรวมค่าบริการเสริม
5. แสดงสรุปการจองและข้อมูลลูกค้า
